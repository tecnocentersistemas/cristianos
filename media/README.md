# Media - FaithTunes

Este directorio contiene todos los recursos multimedia del proyecto.

## Estructura

```
media/
├── images/       Fotos de paisajes y animales (Pexels/Unsplash, licencia libre)
│   ├── mountains/    Montañas, cerros, cumbres, formaciones rocosas
│   ├── rivers/       Ríos, lagos, cascadas, océanos, costas
│   ├── sunsets/      Atardeceres, amaneceres, cielos, estrellas, auroras
│   ├── forests/      Bosques, árboles, senderos, caminos
│   ├── animals/      Águilas, palomas, corderos, leones, ciervos, mariposas
│   └── fields/       Campos, valles, praderas, jardines, cosechas
├── audio/        Instrumentales generados con Suno AI
│   ├── country/
│   ├── rock/
│   ├── gospel/
│   ├── folk/
│   ├── worship/
│   └── ballad/
└── videos/       Videos finales generados con FFmpeg (MP4)
```

## Fuentes de imágenes (licencia libre)
- **Pexels**: https://www.pexels.com/ (100% gratis, sin atribución requerida)
- **Unsplash**: https://unsplash.com/ (100% gratis, sin atribución requerida)
- **Pixabay**: https://pixabay.com/ (100% gratis)

## Formato recomendado
- **Imágenes**: JPG, 1920x1080 mínimo (Full HD), comprimidas para web
- **Audio**: MP3, 192kbps mínimo, 2-4 minutos por track
- **Video**: MP4, H.264, 1920x1080, 30fps

## Naming convention
- Usar snake_case: `nombre_descriptivo_numero.ext`
- Ejemplo: `mountain_sunrise_01.jpg`, `country_gentle_faith_01.mp3`
