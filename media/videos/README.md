# Videos Generados - FaithTunes

Este directorio almacena los videos MP4 finales generados con FFmpeg.
Cada video combina: imágenes + instrumental + versículos sobreimpresos.

## Formato
- Contenedor: MP4
- Codec video: H.264
- Codec audio: AAC
- Resolución: 1920x1080 (Full HD)
- FPS: 30
- Naming: video_id_titulo_idioma.mp4 (ej: 01_montanas_de_fe_es.mp4)
